# 不動産投資 収益シミュレーター

Excel版「収益計算シミュレーション」をベースに、検証で判明した不具合を修正して
Webアプリ化したもの。依存ライブラリなしの静的サイト（HTML/CSS/JS）。

## 機能

- **試算結果**: 35年分の税引後CF・損益・KPI（表面/NOI利回り・CCR・DSCR・自己資金回収年・デッドクロス）
- **出口戦略**: 売却年別のトータル利益とIRR、最有利売却年のハイライト、長期譲渡切替（7年目）表示
- **感度分析**: 金利上昇・空室増・家賃下落・売却価格変動をスライダーで即時再計算
- **諸費用計算**: 仲介手数料（低廉特例対応）・印紙税（軽減税率）・登録免許税・不動産取得税の概算
- 個人（所得税・損益通算）/ 法人（実効税率・欠損金繰越・均等割）切替
- ライト/ダークモード対応・入力値はlocalStorageに自動保存

## Excel版から修正した主な計算ロジック

| 論点 | Excel版 | 本アプリ |
|------|---------|----------|
| 頭金の扱い | 儲けから未控除 | 年0のキャッシュアウトとして控除 |
| 中古耐用年数 | 築古（法定超過）で破綻 | 法定×20%・下限2年に対応 |
| 長期譲渡判定 | 6年目から長期 | 7年目から長期（1/1時点5年超） |
| 給与所得控除 | 旧テーブル＋符号ミス | 令和7年度改正（最低65万円） |
| 法人の赤字年 | 負の法人税（即時還付） | 税額0＋欠損金10年繰越＋均等割 |
| 土地利息の損益通算 | 制限なし | 措法41条の4を適用 |
| 減価償却の最終年 | 簿価超過償却 | 残存簿価調整・備忘価額1円 |
| 端数月の借入期間 | 最終端数年が消える | 月次償還で正確に集計 |
| 印紙税（売買） | 本則税率 | 軽減税率（2027/3/31まで） |
| 入替コスト月割 | 分母の参照ミス | 修正済み |

## デプロイ

静的ファイルのみなので、GitHub Pagesにそのまま置けます:

```bash
cd real-estate-sim
git init && git add -A && git commit -m "initial"
# pinthome組織のリポジトリにpushして Pages を有効化
```

ローカル確認: `python3 -m http.server 8000` → http://localhost:8000

## 前提・免責

- 年初取得・年単位の近似計算（月割りなし）
- 住民税は所得割10%のみ（均等割・調整控除なし）
- 消費税・青色申告特別控除・繰上返済・金利変動は未対応
- 税制は2026年時点。個別の税務判断は税理士に確認のこと
